# 波加曼互動頁面 設計文件

日期：2026-07-18
狀態：已與使用者確認定案

## 目的

做一個波加曼（Piplup）主題的好玩互動網頁：波加曼跟著滑鼠跑、整個頁面是「活的」（背景影片輪播＋手繪塗鴉飄浮裝飾）。用途為自己玩、分享給朋友，不公開營利。

## 總體架構

主力在 Lovable 平台開發（React + Vite + Tailwind，Lovable 預設技術棧），透過 Lovable MCP 建立專案與迭代。完成後由 Lovable 部署成公開網址分享給朋友（按下部署前需再向使用者確認一次）。

採**雙版本**策略處理背景影片的版權問題：

- **線上版**：背景影片用 YouTube IFrame API 嵌入輪播（YouTube 官方允許的使用方式，無散布問題）。代價：角落有 YT 浮水印、手機上必須靜音才能自動播放。
- **本地版**：同一份程式碼，加影片來源開關——偵測到本地 `videos/` 資料夾有影片檔時，改用 `<video>` 標籤播放本地檔案（無浮水印、離線可玩）。影片以 yt-dlp 下載至使用者電腦供個人使用，**不上傳、不進部署**（公開再散布官方影片屬版權越線，明確排除在範圍外）。

## 元件切分

| 元件            | 職責                                                                                                                                                                                                                           |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `VideoBackdrop` | 全頁最底層背景影片輪播。線上版走 YouTube IFrame API（靜音自動播、cover 滿版裁切、播畢自動切下一部）；本地版走 `<video>` 標籤輪播本地檔。清單先放 3–5 部官方波加曼影片，可隨時增改。上方疊一層淡藍半透明濾鏡，確保前景清楚。    |
| `DoodleLayer`   | 前景手繪塗鴉裝飾層：手繪風內嵌 SVG（水泡、星星、小魚、音符）以 CSS animation 緩慢飄移、旋轉、忽大忽小，各元素速度不同以營造層次；滑鼠移動時偶爾灑落小水滴粒子。此層不攔截滑鼠事件（pointer-events 穿透），避免干擾波加曼跟隨。 |
| `Piplup`        | 跟隨滑鼠的波加曼本體，內含三狀態機（見下）。素材用 PokeAPI 公開 sprite／官方 artwork 圖檔網址。                                                                                                                                |

## 波加曼行為狀態機

- **追逐（chase）**：`requestAnimationFrame` + 緩動插值（lerp）追滑鼠，離游標越遠速度越快，靠近後停下。跑動時依方向左右翻轉、身體小幅上下彈跳。
- **發呆／睡著（idle → sleep）**：滑鼠約 5 秒不動進入發呆（左右張望、眨眼）；約 15 秒進入睡著（頭上飄手繪「Zzz」）。滑鼠一動立即回到追逐。
- **點擊反應（react）**：點到波加曼時跳一下、冒出手繪愛心／泡泡，隨後回到追逐。

## 視覺風格

淡藍水系配色（呼應波加曼），手繪塗鴉風前景。背景影片為主視覺，塗鴉層為前景裝飾框住畫面。

## 錯誤處理

- YouTube 影片無法載入（下架、地區限制）時自動跳下一部；全部失敗則退回純塗鴉背景（淡藍紙張質感底色）。
- 本地版 `videos/` 為空或檔案無法播放時，退回 YouTube 嵌入模式。
- PokeAPI 圖檔載入失敗時以內嵌 SVG 簡筆波加曼替代，互動不中斷。

## 驗證方式

無測試框架，改以瀏覽器實測：開啟 Lovable 預覽頁，逐一驗證（1）滑鼠跟隨與翻轉、（2）點擊反應、（3）發呆／睡著與驚醒、（4）影片輪播與切換、（5）塗鴉層動畫與粒子，並截圖回報。本地版另驗證本地影片播放與離線開啟。

## 範圍外（明確排除）

- 將下載的官方影片上傳至任何公開部署。
- 心情值、分數等遊戲化系統（YAGNI）。
- 音效（未提出需求）。
